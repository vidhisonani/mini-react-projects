# ☕ Coffee Cafe — Modern Cafe Website (React)

Coffee Cafe is a **frontend React project** that represents a modern coffee shop website with smooth animations, a clean UI, and responsive design.

The project focuses on building a visually appealing café landing page using **React**, **Tailwind CSS**, and animation libraries to enhance user experience.

This project was created as part of my **frontend & React learning journey**, with a focus on:
- Component-based UI design  
- Animations and user interaction  
- Responsive layouts  
- Clean styling using Tailwind CSS  
- Theme management (Light / Dark mode) 

**Live Demo:** [Coffee cafe](https://coffee-cafe-by-vidhi.netlify.app/)

---

## 🚀 Features

- Modern café-themed UI  
- Dark Mode / Light Mode toggle  
- Smooth scroll animations using **AOS**  
- Responsive design for all screen sizes  
- Reusable React components  
- Icons using **react-icons**  
- Clean and minimal layout  
- Fast and lightweight frontend  

---

## 🛠 Tech Stack

## 🔧 Setup & Run Locally

1. **Clone the repository**

```bash
git clone https://github.com/vidhisonani/mini-react-projects.git
```

2. Go into the project folder:

```bash
cd 05-coffee-cafe
```

3. **Install dependencies**
```bash
npm install
```
4. **Start the development server**
```bash
npm run dev
```
